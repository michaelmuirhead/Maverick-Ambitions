# Maverick Ambitions

A life and business simulation desktop game — think Big Ambitions meets BitLife meets The Sims.

## Development

```bash
npm install
npm run dev       # launch with hot-reload
npm run build     # production build
npm run typecheck # type-check all files
```

## Stack

- Electron 33 (desktop shell)
- React 18 + TypeScript
- Vite via electron-vite
- CSS Modules
